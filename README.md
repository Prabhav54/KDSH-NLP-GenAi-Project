## 📖 Overview

This repository contains the official codebase and modeling pipeline developed for the **Kharagpur Data Science Hackathon (KDSH) 2026**. It leverages state-of-the-art Natural Language Processing (NLP) techniques and Generative AI models to solve complex, data-driven challenges. The project demonstrates an end-to-end data science lifecycle, from data ingestion and preprocessing to model training and a deployment-ready architecture.

## ✨ Key Features

- **Advanced NLP Pipeline:** Efficient tokenization, embedding, and semantic processing of textual data.
- **Generative AI Integration:** Implementation of modern LLMs/GenAI techniques for text generation, summarization, or classification.
- **Scalable Architecture:** Modular code design utilizing the `src/` directory as a package with standard `setup.py` configurations.
- **Interactive Exploration:** Comprehensive Jupyter Notebooks detailing Exploratory Data Analysis (EDA) and experimental model iterations.

## 🗂️ Repository Structure

The project directory is structured to ensure clean code practices and reproducibility:
